Section 1

Responsive web design is an approach that ensures all the pages of the website look, work and feel perfectly on any device. Whether it is a tiny old cell phone with a screen width of 320px, a modern phablet with 7 inches screen, a big iPad, or a TV with a massive diagonal line, all the main aspects such as content, design, and especially functionality should perform consistently to provide users with an excellent user experience.

Section 2

The main benefits of using a responsive website are

    consistent visual experience;
    better user experience;
    no need for redirects;
    lower bounce rates;
    lower maintenance needs;
    high web page loading speed;
